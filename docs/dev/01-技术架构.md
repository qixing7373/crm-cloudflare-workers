# 粉库系统 — 技术架构

> **技术栈**：Hono + Drizzle ORM + Cloudflare D1 + Vue 3 + Naive UI + Rsbuild + Bun

---

## 架构总览

```
┌──────────────────────────────────────────────┐
│              Cloudflare Pages                │
│           Vue 3 + Naive UI (前端)            │
│         部署目录：web/                         │
└──────────────┬───────────────────────────────┘
               │ fetch /api/*
┌──────────────▼───────────────────────────────┐
│           Cloudflare Workers                 │
│         Hono (API Server)                    │
│         Drizzle ORM (查询层)                  │
│         部署目录：api/                         │
└──────────────┬───────────────────────────────┘
               │ D1 Binding
┌──────────────▼───────────────────────────────┐
│           Cloudflare D1                      │
│         SQLite (数据库)                       │
└──────────────────────────────────────────────┘
```

**部署模型**：全部运行在 Cloudflare 边缘网络，零服务器运维。

**项目结构**：

## 技术选型理由

| 组件 | 选型 | 理由 |
|------|------|------|
| API 框架 | **Hono** | 超轻量、TypeScript 原生、Workers 原生支持 |
| ORM | **Drizzle** | 类型安全、D1 原生支持、自带 migration + studio |
| 数据库 | **Cloudflare D1** | Serverless SQLite、免运维、本地开发即 SQLite 文件 |
| 前端框架 | **Vue 3 + Naive UI** | 组件覆盖完整，按需加载，和当前代码实现一致 |
| 构建工具 | **Rsbuild** | 基于 Rspack，比 Vite 更快，Vue 插件支持 |
| 包管理 | **Bun** | 安装速度快，替代 npm/node |
| 部署 | **Cloudflare Workers + Pages** | 全球边缘、零运维、免费额度充足 |

## 本地开发环境

```bash
# 初始化项目
cd api && bun install
cd web && bun install

# 本地开发
cd api && bun run dev       # Wrangler dev server (localhost:8888)
cd web && bun run dev       # Rsbuild dev server (localhost:7777)

# 数据库
cd api && bunx drizzle-kit generate   # 生成 migration
cd api && bunx drizzle-kit studio     # 数据库可视化

# 部署
cd api && bun run db:migrate:remote                   # D1 migration
cd api && bun run db:seed:remote                      # 初始化字段和 admin
cd api && bunx wrangler deploy                         # 部署 API Worker
cd web && bun run build                                # 构建前端
# 前端通过 Cloudflare Pages 自动部署（连接 Git 仓库）
```

## 开发计划

| 阶段 | 天数 | 内容 |
|------|:----:|------|
| **D1** | 1天 | 项目初始化、Drizzle schema、D1 建表、wrangler 配置 |
| **D2-D3** | 2天 | 认证系统（注册/登录/JWT）、用户管理 API |
| **D4-D5** | 2天 | 核心 API：手机号查询、标记转化、批量导入去重（含变更追踪） |
| **D6** | 1天 | 动态字段 CRUD API + 字段管理页面 |
| **D7-D8** | 2天 | 前端：登录/注册页、业务员查询页 |
| **D9-D10** | 2天 | 前端：管理员导入页、数据列表页、用户管理页 |
| **D11** | 1天 | 转化统计（按月）、系统配置页 |
| **D12** | 1天 | i18n 多语言（中/英） |
| **D13-D14** | 2天 | 安全渗透测试、联调测试、优化、部署上线 |

**总计约 14 个工作日（~3 周）**
